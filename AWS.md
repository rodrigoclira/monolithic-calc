# Executar na AWS (Cloud9)

1. Realize a atualização do repositórios e instalação dos pacotes necessários

```
sudo apt-get update
sudo apt-get install python3-venv unzip -y
```

2. Faça o download do repositório
```
wget  https://github.com/rodrigoclira/monolithic-calc/archive/refs/heads/main.zip
```

3. Descompacte repositório na pasta atual
```
unzip main.zip
```

4. Acessar pasta do repositório

```
cd monolithic-calc
```

5. Volte para o README.md
